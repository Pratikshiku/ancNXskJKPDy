### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Visual Studio Code(VScode)`
- `Navicat`

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 课程信息
5. 学习论坛
6. 试卷
7. 公告资讯
8. 后台管理
9. 个人中心
10. 我的发布
11. 考试记录
12. 错题本
13. 我的收藏


**教师功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 课程信息管理
4. 课程信息
5. 试卷管理
6. 试题管理
7. 考试管理（考试记录、错题本）



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 学生管理
4. 教师管理
5. 班级管理
6. 课程类别管理
7. 课程信息管理
8. 学习论坛
9. 试卷管理
10. 试题管理
11. 系统管理（关于我们、系统简介、轮播图管理、公告资讯）
12. 考试管理（考试记录、错题本)
